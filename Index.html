Name Type Example Usage
Table element(Tag)type selector <table> The wrapper element for all HTML tables
Table Head element(Tag)type selector <thead> The set of rows defining the column headers in a table
Table Body element(Tag)type selector <tbody> The set of rows containing actual table data
Table Row element(Tag)type selector <tr> The table row container
Table Data element(Tag)type selector <td> The table row container
Table Foot element(Tag)type selector <tfoot> The set of rows defining the footer in a table
Colspan Attribute tag colspan Defines how many columns a td element should span
Rowspan Attribute tag rowspan Defines how many rows a td element should span
